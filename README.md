# Installation
```bash
$ npm install oromis --save
```

## Preview

```javascript
import Oromis from 'oromis';

Oromis.uuid();
// 1c471c5e-7260-4d42-a625-1cd83f66bf03

Oromis.uuid({ uppercase: true });
// 18633212-D519-42DC-B0E7-D02441684CEC
```
